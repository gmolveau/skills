# skills by gmolveau

## Getting started

1. Run the skills.sh installer:

```bash
npx skills@latest add gmolveau/skills
```

2. Pick the skills you want, and which coding agents you want to install them on.

## The skills

- `validate-idea` : "grill-me" style to pressure-test project ideas, not technical
